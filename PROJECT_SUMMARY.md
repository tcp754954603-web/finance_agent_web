## 🔧 核心技术栈

### 后端技术
- **Flask**: Web框架和API服务
- **yfinance**: Yahoo Finance数据获取
- **pandas**: 数据处理和分析
- **ta**: 技术分析指标库
- **numpy**: 数值计算
- **requests**: HTTP请求处理

### 前端技术
- **HTML5/CSS3**: 现代化界面设计
- **JavaScript ES6+**: 交互逻辑实现
- **Chart.js**: 专业图表渲染
- **响应式设计**: 多设备适配

### 数据源
- **Yahoo Finance**: 免费实时股票数据
- **Alpha Vantage**: 专业金融数据API
- **市场指数**: 主要股指实时数据

## 📈 使用示例

### 启动服务器
```bash
# 使用默认配置
python app.py

# 使用自定义参数
python app.py --host 0.0.0.0 --port 12000 --no-debug

# 使用启动脚本
python start_server.py --port 12000
```

### API调用示例
```bash
# 获取股票数据
curl "http://localhost:12000/api/stock_data?symbol=AAPL&period=1mo&analysis_type=quick"

# 获取市场概览
curl "http://localhost:12000/api/market_overview"
```

### 功能演示
```bash
# 运行演示脚本
python demo.py
```
