# 🚀 智能理财炒股 Agent - 专业版

一个功能强大的股票分析平台，集成了多数据源获取、技术分析、AI智能推理等功能。

## ✨ 主要功能

### 📊 数据获取
- **多数据源支持**: 集成 yfinance 和 Alpha Vantage API
- **多时间框架**: 支持1分钟到2年的各种时间周期
- **实时数据**: 获取最新的股票价格和市场数据
- **市场概览**: 显示主要股指的实时表现

### 🔍 技术分析
- **技术指标**: RSI、MACD、布林带、移动平均线等
- **支撑阻力**: 自动识别关键价格水平
- **交易信号**: 基于技术指标生成买卖信号
- **趋势分析**: 识别价格趋势和反转点

### 🤖 AI智能分析
- **多模式分析**: 快速分析、技术分析、完整AI分析
- **综合评估**: 结合技术面和基本面数据
- **中文报告**: 生成详细的中文分析报告
- **投资建议**: 提供基于数据的投资建议

### 📈 可视化界面
- **专业图表**: 使用 Chart.js 创建交互式图表
- **响应式设计**: 支持桌面和移动设备
- **实时更新**: 动态加载和更新数据
- **多面板布局**: 清晰展示各类信息

## 🛠️ 技术架构

### 后端技术
- **Flask**: Web框架
- **yfinance**: 股票数据获取
- **pandas**: 数据处理
- **ta**: 技术分析库
- **requests**: HTTP请求

### 前端技术
- **HTML5/CSS3**: 现代化界面
- **JavaScript**: 交互逻辑
- **Chart.js**: 图表渲染
- **响应式设计**: 适配各种设备

### 数据源
- **Yahoo Finance**: 免费股票数据
- **Alpha Vantage**: 专业金融数据API
- **实时市场数据**: 主要股指和个股

## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 配置设置
编辑 `config.yaml` 文件：
```yaml
# Alpha Vantage API配置（可选）
alpha_vantage:
  api_key: "YOUR_API_KEY"
  base_url: "https://www.alphavantage.co"

# LLM配置（可选）
llm:
  enable: false  # 设为true启用AI分析
  model: "qwen2.5:7b"
  base_url: "http://localhost:11434"

# 服务器配置
server:
  host: "127.0.0.1"
  port: 5000
  debug: true
```

### 3. 启动服务器
```bash
# 使用默认配置
python app.py

# 或指定参数
python app.py --host 0.0.0.0 --port 12000 --no-debug
```

### 4. 访问应用
打开浏览器访问: `http://localhost:12000`

## 📋 使用说明

### 基本操作
1. **输入股票代码**: 如 AAPL、TSLA、NVDA等
2. **选择时间周期**: 1天到2年的各种选项
3. **选择数据间隔**: 1分钟到1天的间隔
4. **选择分析类型**: 快速、技术或完整AI分析
5. **点击分析**: 获取综合分析结果

### 功能面板
- **市场概览**: 查看主要股指表现
- **股票信息**: 显示公司基本信息
- **价格走势**: 交互式价格图表
- **技术指标**: RSI、MACD、成交量等
- **技术分析**: 详细的技术指标解读
- **交易信号**: 基于算法的买卖建议
- **AI分析**: 智能分析和投资建议

## 🔧 高级配置

### Alpha Vantage API
1. 注册 [Alpha Vantage](https://www.alphavantage.co/) 账户
2. 获取免费API密钥
3. 在 `config.yaml` 中配置API密钥

### AI分析功能
1. 安装 [Ollama](https://ollama.ai/)
2. 下载 Qwen2.5 模型: `ollama pull qwen2.5:7b`
3. 在配置文件中启用LLM功能
